# Common Tools
- Ghidra
- Radare2
- Ltrace
- Strace
- gdb

## Gihdra
- Debugger
- Decompiler
- Graphic Interface

## Radare2 
- `r2 -d /path/to/file`
- `aaa` analyze file
- `afl` list functions
- `s function_name`
- `pdf` print assembly code of the function

## ltrace
- `ltrace /path/to/file`

## GDP
- `gdp /path/to/file`
- `info functions`
- `b *pointer_value` breakpoint
- `run test`
- `info registers`
